# 🤖 Mindfriend - Asistente de Apoyo Emocional

Mindfriend es un chatbot inteligente diseñado específicamente para acompañar a estudiantes en momentos de estrés o soledad. Utiliza inteligencia artificial local para ofrecer un espacio seguro, privado y empático.

## ✨ Características Principales
- **Diseño Neumórfico:** Interfaz visual moderna, relajante y adaptada a dispositivos móviles.
- **Privacidad Total:** Gracias a **Ollama**, el procesamiento de datos ocurre 100% en la computadora del usuario.
- **Protocolo de Seguridad:** Sistema de detección de crisis que muestra alertas con contactos de emergencia (Línea de la Vida).
- **Memoria Contextual:** Capacidad de recordar el hilo de la conversación para una interacción más humana.

## 🛠️ Requisitos e Instalación
1. **Ollama:** Descargar e instalar desde [ollama.com](https://ollama.com).
2. **Modelo:** Ejecutar en terminal `ollama pull gemma3:1b`.
3. **Librerías:** Instalar dependencias con `pip install flask requests`.

## 🚀 Ejecución
1. Ejecutar el servidor: `python app.py`.
2. Abrir en el navegador: `http://127.0.0.1:5000`.
